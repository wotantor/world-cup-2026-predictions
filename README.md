## 📜 Правила начисления баллов
* **3 балла** (`🟢`) — абсолютно точно угаданный счёт матча (например, факт 2:1 и прогноз 2:1).
* **1 балл** (`🟡`) — угадан исход или разница мячей (например, факт 2:1, а прогноз 1:0).
* **0 баллов** (`⚪️`) — исход матча не угадан.
