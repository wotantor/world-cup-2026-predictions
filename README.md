## 📜 Правила начисления баллов
* **3 балла** (`🟢`) — абсолютно точно угаданный счёт матча (например, факт 2:1 и прогноз 2:1).
* **1 балл** (`🟡`) — угадан исход или разница мячей (например, факт 2:1, а прогноз 1:0).
* **0 баллов** (`⚪️`) — исход матча не угадан.

## 🛠 Как заколлабиться и зафиксировать свой прогноз (Защита от читерства)
Чтобы никто не изменил свой прогноз в Google Таблице задним числом, мы фиксируем ставки через Git:

1. Сделайте **Fork** этого репозитория.
2. В папке `predictions/` создайте текстовый файл со своим именем (например, `sanya.txt`).
3. Впишите туда свои прогнозы на ближайший тур и создайте **Pull Request** (PR) до начала первого матча тура.
4. После апрува ваш прогноз навсегда зафиксируется в истории коммитов GitHub со штампом времени (Timestamp).
